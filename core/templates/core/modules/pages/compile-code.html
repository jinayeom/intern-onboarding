Compiling Your Code

In order to compile the changes you made in your code, take a look at the build.sh script. This script contains the sequential commands that need to be run so you can compile your code.

Follow these steps to actually compile your code:

  1. In your terminal, change directory to CE_WebApps directory, not the CE_WebApps/CE_WebApps folder: cd ..
  2. Run the build.sh script: ./build.sh
  3. Navigate to your local sandbox browser to confirm the changes have taken effect.
