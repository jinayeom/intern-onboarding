Overview

TCP/IP is the protocol suite that governs how data moves across the Internet. You’ll learn about IP addressing, routing, TCP handshakes, and how packets travel from your computer to a destination server.

Estimated time: 30 minutes
Difficulty: Beginner

IP

  • Basic understanding of computer networks
  • Familiarity with IP addresses

Guided steps

  1. Use ping google.com to test connectivity.
  2. Run traceroute google.com to trace packet hops.
  3. Observe TCP connections with netstat -an.

Cheatsheet

# Show routing table
netstat -rn
# Trace path to host
traceroute example.com
# View open ports
lsof -iTCP -sTCP:LISTEN
